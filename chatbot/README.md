# チャットボット for LINE & Facebook

### 必要な設定

認証情報
* cloudant_credentials.json
* conversation_credentials.json
* conversation_workspace_id.json
* facebook_api_credentials.json
* line_api_credential.json

SSL通信用証明書
* lets_encript.key
* lets_encript_fullchain.crt

npmパッケージ設定
* package.json

必要な環境
* node v4.4.7

$ npm install
